
# Project Overview

Web-based application that reads RSS feeds. This project demonstrates browser based testing using [Jasmine](https://jasmine.github.io/).

# How to run this project

- Install http-server via `npm`

	`npm install http-server -g`
- Run `Git clone https://github.com/mangooose/frontend-nanodegree-feedreader.git`
-  cd frontend-nanodegree-feedreader/
- Run `http-server`
- Go to localhost:8080 in your browser
